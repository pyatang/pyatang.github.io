DOSKEY定义命令宏

一次性别名 dosky gad=git add $* $* 表示后面还有参数

批处理脚本(每次启动cmd时执行)

1.写一个批处理脚本每次启动cmd时自动执行需要初始化的别名 2.win+r，键入regedit，进入地址： 计算机\HKEY_CURRENT_USER\Software\Microsoft\Command Processor 3.右键点击Command Processor新建一个字符串 AutoRun 4.右键点击AutoRun，数值设置为每次启动时要执行批处理脚本。 例如：AutoRun f:\scripts\ly-alias.bat 

@echo off
doskey gad=git add $*
doskey gcm=git commit -m $*
doskey gph=git push $*
doskey gpl=git pull $*
