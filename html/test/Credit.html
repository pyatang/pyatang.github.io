授信流程分析

测试数据的准备工程，相当于对整个服务的逆向思考。 授信过程涉及到的点
  1. 获取用户的基本信息，包含姓名、身份证
    • 用函数生成伪造姓名
      • 使用mysql 的 procedure存储函数来生成字符串和数字类型的数据 
      CREATE PROCEDURE testcases.`test-insert`(in len int)
begin
	declare x int;
set x=1;
	while x<10 do
	insert into bankcard (age,number) values(x,x);
set x = x +1;
	end while;
select * from bankcard b limit len;
END
    • 用函数生成伪造手机号
    • 用函数生成伪造身份证——符合格式，能通过程序校验
    • 用函数生成伪造银行卡——符合格式，能通过程序校验
    直接插入数据库表 passport.user_base
  2. 携带用户信息发起请求
    • 明确注册调用的接口
  3. 得到请求结果
